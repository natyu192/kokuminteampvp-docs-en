Team Deathmatch
==================

**Team Deathmatch** モードでは、制限時間内のチーム合計キル数を競います。

================= ====================================== ====== ============
項目(\*は必須)    説明                                   値     デフォルト値
================= ====================================== ====== ============
maxscore          この得点に到達すると試合が終了します。 数値   100
================= ====================================== ====== ============
